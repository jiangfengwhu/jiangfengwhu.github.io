CentOS安装

  • 系统装在SSD，机械硬盘用作网站静态文件存储
  • 镜像选择minimal最小安装
  • 分区
    • /boot 400M
    • /boot/efi 200M
    • swap 2G (内存16G，目前很少用到swap，以后用得到再加)
    • / 其余
  • 网络
    • 超五类网线(千兆)连接
    • eno1设置所有用户可用
    • 连接可用时自动连接

机械硬盘挂载

首先查看机械硬盘设备

1

sudo fdisk -l

然后利用fdisk对硬盘分区

1

sudo fdisk /dev/sda

此处只分一个区，一路enter，之后对/dev/sda1进行格式化(xfs)。

1

sudo mkfs.xfs -f /dev/sda1

格式化完成之后将其挂载在/disk1。

1

sudo mount /dev/sda1 /disk1

设置开机挂载

1
2

vi /etc/fstab
/dev/sda1  /disk1  xfs  defaults     0   0

  • epel-release
  • bash-completion-extras

  • devtoolset-8

    • 如需设置成默认，添加下面内容到/etc/profile.d/devtoolset-8.sh:

2

      #!/bin/bash
source scl_source enable devtoolset-8

  • yum-utils

  • cmake3

  • git

  • wget

  • ffmpeg 点此下载

vim编译

CentOS自带vim版本过旧，下载vim源码，之后进入源码目录。

  • 安装依赖

    1

    sudo yum install ncurses-devel python-devel

  • 编译vim源码

    1
2
3

    ./configure --with-features=huge --enable-gui=no --disable-nls --enable-cscope --enable-multibyte --enable-pythoninterp
make 
sudo make install

vim安装完毕，推荐一个vim配置。

Nginx配置

设置nginx.conf中的user为root，根据你的需求配置nginx。

1

1

1

SELinux相关

 drwxr-xr-x xxxxxx

如果开启了SELinux，请注意将上述路径进行如下设置：

1
2

getenforce \\查看是否开启SELinux
chcon -Rt httpd_sys_content_t /path/to/www

同时，若nginx内部设置了反向代理，需进行如下设置：

1

setsebool -P httpd_can_network_connect on

Let’s Encrypt证书相关

给网站申请免费的SSL证书，参考Let’s Encrypt官网。

首先安装certbot

1

sudo yum install certbot

采用standalone方式申请证书需要占用80端口，需要先停止nginx

1

sudo service nginx stop

执行

1

sudo certbot certonly --standalone -d youdomain

按照要求填写相关信息即可。

生成4096位的DH-Key，增强网站安全性

1

sudo openssl dhparam -out /path/to/dhparam.pem 4096

最后在nginx配置文件中配置https即可。

至此，nginx配置基本完成。

P2P相关

Tracker

网站视频部分添加了p2p模块，需要tracker交换SDP信息，出于安全和性能考虑，没有使用public tracker。目前使用nodejs版的tracker。安装如下

1

npm install -g bittorrent-tracker

TURN Server

同上，自建TURN服务器，使用的是coturn，为了方便，并没有编译安装coturn，而是使用的官方给的预编译文件点此下载，如需使用最新版，请自行编译。

1

sudo turnserver -a -o -v -n  --no-dtls --no-tls -u xxx:xxx -r "xxx"
